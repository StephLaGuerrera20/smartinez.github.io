Calculus I - L'Hopital's Rule

L'Hôpital's Rule:

Using the concept of a limit, we are able to derive the derivative, also known as the
instantaneous rate of a function, of any point on a graph. As change in x becomes
infintesimally small we know that the equation for slope approaches the rate of change
at a given point. Using the derivative, we are able to interpret the nature of a graph.

But, how do we know what the limit of a function is if it results in an indeterminate form
when x is plugged into the function?

Example:

When 0 isplugged in for x, it results in the indeterminant form, 0 over 0,
making the limit unsolvable.

The answer lies within L'Hôpital's Rule which is repeatedly taking the derivative of both
functions in the numerator and denominator. There are three instances where L'Hôpital's Rule
can be used which is when the function is equal to:

  • 0 over 0
  • Positive infinity over infinity
  • Negative infinity over infinity

Going back to the example, we can find the derivative of both the numerator and denominator
to get the limit.
